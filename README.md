## Installation

* Create a virtual environment:
   ```bash
   python -m venv myenv
  
* Activate the virtual environment:
    * For Windows:
   ```bash
  env\Scripts\activate
   ```
  * For Unix or MacOS::
     ```bash
     source myenv/bin/activate
     ```
* Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
* Run the application:
   ```bash
   python manage.py runserver
   ```
